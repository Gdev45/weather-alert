# Weather Alerts CLI

Terminal tool for live weather warnings (UK & USA), color-coded and easy to read.

## Installation

You can install the CLI with `pipx` (recommended):

```bash
pipx install git+https://github.com/Gdev45/weather-alert.git


```
## How to run

```bash
weather-alerts
```
